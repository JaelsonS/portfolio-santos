# Portfolio — Jaelson Santos

Este é o meu portfólio pessoal. Ele é leve, direto e pensado para recrutadores. Hoje o projeto está no **root** (HTML/CSS/JS puro).

## O que eu quis mostrar aqui

- Layout moderno, 100% responsivo e acessível.
- Navegação clara com menu mobile.
- Redes sociais completas com ícones.
- Formulário com validação simples e fallback para email.

## Stack que usei

**Frontend**
- HTML5, CSS3, JavaScript (ES6+)
